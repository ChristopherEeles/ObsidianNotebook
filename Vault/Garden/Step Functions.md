# Step Functions
* Also called activation functions, are applies to an neural network output to determine the state of the artifical neural circuit

## Heaviside Function
$$ \large
heaviside(z) = \begin{cases}
	0 \text{ if } z \lt 0 \\
	1 \text{ if } z \ge 0
\end{cases}
$$
## Sign Function
$$\large
sign(z) \begin{cases}
-1 \text{ if } z \lt 0\\
0  \text{  if } z = 0\\
1 \text{  if } z \gt 0
\end{cases}
$$
