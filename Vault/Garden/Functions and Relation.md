# Functions and Relations
## Relation
- A relation is a set of ordered pairs
	- Let $A = \{(x_1, y_1), (x_2, y_2), ..., (x_n, y_n)\}
	- Let $A = \{(1,2), (2,4), (3, 6)\}$

## Function
- A function is a relation in which for every value of $x$ there is only one value of $y$
